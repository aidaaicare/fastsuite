FASTSUITE Edition 2 has a main window, one single user interface, that is called the 3D View. This 3D View not only displays the 3-dimensional model, it integrates all user interactions and all other command toolbars. The 3D View is the tool that visualizes, interacts with, analyzes and simulates the models and layouts that the user is working on.

User interaction with the 3D View is supported with the common mouse and keyboard workstation devices.

The 3D View has several interactive areas, where certain commands or actions are started. The 3D View itself is the environment where all the direct interactions with the 3D objects are taking place.

Interactive zones in the 3D View

 1 Workbench carousel

 2 Document toolbar

 3 Operation mode

 4 View toolbar

 5 View Compass

 6 Workbench toolbar

 7 Workbench dashboard

 8 Headup display

 9 Help display

10 Simulation player

11 Active frame

12 Display filter

13 Console

 

The 3D View has two workspaces: the Show and Hide workspace.

 

The 3D View supports many different UI interactions and embeds several general commands.
