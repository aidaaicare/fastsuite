Program toolpath


The Program toolpath command can be executed on point based process geometry.


Existing PG points can be picked with the left mouse button, supported with the available multi-select possibilities. The selected points are highlighted. In between them a dashed, arrow symbolized line is displayed to indicate the motion direction along the path. Thus, the sequence of selecting the process geometry objects automatically determines the travel route of the tool path.


While the selection is still active and no other commands have been executed, the selection of points can be modified. Several possibilities are supported.

Add a selection after last position

To add a new process geometry after the last position of the path is simply done by continuing the selection. The new selection automatically is added after the last step.

Add a selection between two existing positions

To add a position between two existing ones can be achieved with the dashed path line. This line can be picked with the left mouse button and dragged on top of the new process geometry. This process geometry then has to be selected with the keyboard button pressed. It will be inserted in the current path, in between the two positions that were connected by this path line.



Removing a selection

To remove a selected process geometry object from the tool path can be simply done. With the keyboard button pressed and picking the object again it will be removed from the current path.

 


Add a selection before the first position

To add a process geometry before the first position of the current path is a combination of adding a selection and removing a selection. The new process geometry first will be added after the current first position by dragging the path line to it. It will become the second position after this step. Then the first position will be removed and added again.



On the selection a Pie menu can be opened:

Activate Manipulator

Insert operation

Select process geometry

Program process geometry

Sequencing

After computation the tool path is displayed. Point based process geometry generates a tool path element at each of the selected process geometry points.
