Partial simulation

Teach


With the Partial simulation command a user defined part of the toolpath will be simulated.

When the command has been started on a TPE, this TPE will be the starting point for simulation. Another TPE has to be selected to where the simulation has to run. After that the simulation will be played.

When the command has been started in on a program, operation group or operation (in the OLP program dashboard), the partial simulation will run that selection.

The Play button in the Simulation player changes automatically to the Partial player mode to be able to re-run the simulation on the defined part of the toolpath.
