Axis system

CAD


The Axis system command does not execute a direct function when it is started. It prepares the 3D View to create or modify axis systems.


There are two ways to create the axis system:

oAt the center of the Manipulator.

oOn a geometrical object.


In both cases the creation Pie menu can be opened.

The result of the execution is the axis system object.
