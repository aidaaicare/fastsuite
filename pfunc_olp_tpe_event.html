With the Event command new events can be created and existing events can be modified.

When the command has been started, the events dialogue window appears, like in the example below.


The available instructions, standard and technology specific, are listed in the top section of the dashboard. The instruction can be applied to be executed before the tool path position has been reached, or after it has left the position. By picking the instruction and dragging it into the Before or After section, the event has been created. The picture below shows an example. Multiple instructions can be assigned at one tool path element.


Each event can have attributes to set a value and behavior to the instruction. These attributes can be opened by clicking on the small triangle in front of the instruction. The value for the instruction then can be modified.


Hovering over the instruction name will display a trash button at the end of the line. Pressing the button will remove the instruction from the list.


Multiple event symbols may be shown at the tool path element.
