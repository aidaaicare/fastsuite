Simulation monitor

OLP


The Simulation monitor opens an information window that runs parallel with the simulation. It shows the operation state of the manufacturing resources, such as the tool position, axis motion, signal state and elapsed simulation time, like the example below. It has no edit functionality.


In the monitor following information is displayed.

Simulation time

The elapsed simulation time of the program.

Active controller

The top line displays the active running controller. When the simulation runs multiple controllers they are listed here. Another controller can be selected here to monitor its running program.

 

 

Singularity status

The singularity status of the production resource is written here. In case of issues the status returns with a corresponding message, like in the example below:

Motion time

The elapsed motion time to the next toolpath element.

 

Program name

The name of the simulated program.

Operation group

 

Operation

 

Tag

The name of the toolpath element (tag) that the TCP is approaching.

Feed rate

The velocity of the TCP.

Accuracy

The accuracy to approach the tag.

 

 

TCP coordinates

The coordinates and rotation of the TCP during the simulation. All referred to the Base frame.

Configuration

The configuration of the production resource.

Turn

The turn of the production resource.

 

 

Joints

The joint axes values during simulation.

Joints


Each joint is displayed in this section of the monitor. In front its name is placed. The horizontal bar with the numerical values represents the motion limits of that joint.

During the simulation the black rectangle indicates the current value of the joint. That value is also written at the right side. The rectangle will slide along the bar when the joint is moving.


With the rectangle sliding along the bar, a green bar appears behind the gray one. This green bar indicates the maximum range that the joint has reached during the simulation (so far). Hovering over this green bar will display this reach in numerical values, as in the example below.
