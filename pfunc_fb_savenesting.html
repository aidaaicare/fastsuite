Save nesting

Fixture Builder

 

With this Save nesting command from the function toolbar, the nesting sheets will be stored on disk as a workpiece type .cenwpc file, ready for re-use for manufacturing. A file browser will be opened to define the location where to store the file and under what name. The initial file name is the default name that has been given to the workpiece at creation, but it can be modified before saving the document.

Furthermore the command supports to possibility to save the nesting in 2D DXF format. The format can be selected from the drop down list in the option Save as type.
