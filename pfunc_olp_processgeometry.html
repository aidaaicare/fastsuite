Process geometry

OLP


With the Process geometry command the existing process geometry can be modified and new process geometry can be created..

When starting the command in the OLP workbench, automatically the existing process geometry of the workpiece(s) is displayed, but only that process geometry type (point, curve or surface) that correlates with the currently set technology.

When saving the project, the new created process geometry and modified process geometry will be saved in the workpiece .cenwpc file.
