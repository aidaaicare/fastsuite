Display filter

3D View


The Display filter command opens a set of filter commands to toggle specific 3D features between the show and hide workspace. It will automatically grabs all the features of that specific group of objects. When the group of features is shown, the corresponding toggle icon is highlighted (green background color).


There is a display filter for:

Skeleton

Adapters

Cameras

Frames

Inactive elements

Toolpaths

Object names

Toolpath optimizations

Special features

Process geometry

Operation


To be able to toggle multiple different objects in a same operation, the Display Filter menu remains open after each toggle operation until it is closed by the user himself through any mouse action outside the menu bar.

The window is of the style dashboard and can be dragged to any desired location on screen and can be re-sized to any preferred size.
