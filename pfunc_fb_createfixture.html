When the Create command in the toolbar is started, the system automatically activates the Manipulator in the 3D View. The Manipulator now drives the fixture design.

The UV plane of the Manipulator is the reference plane, where the base plate of the fixture will be computed. In other words: the distance between the UV plane and the workpiece defines the global height of the fixture. The U-axis of the Manipulator is the reference for the orientation of the base plate.

Based upon the system predefined parameters the fixture consists of a base plate with table mounting holes at each corner and 2 equally distributed workpiece support plates in both length and width direction, being the Manipulator's U- and V-direction. The base plate size is determined from a bounding box around the workpiece.


The fixture is a regular resource type of object. This means that with the use of the Resource properties dashboard several attributes of the fixture can be defined or modified.
