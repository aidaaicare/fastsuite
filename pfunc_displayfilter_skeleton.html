Display skeleton

Display filter


A kinematic skeleton, the description of what parts of the object can move in a certain way, can be assigned to a resource. This skeleton has a graphical representation. For convenience the kinematic skeleton visualization can be switched off with the Show the skeletons button. In contradiction to the Swap Show/Hide operation, this command does not switch between workspaces, nor does it move the skeleton between workspaces. It only displays or hides the complete kinematic skeleton in the current workspace, irrelevant if other objects, such as the attached geometry, are shown or hidden.
