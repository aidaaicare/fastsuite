Program toolpath

 


With the Insert operation command a toolpath is generated on the selected process geometry.

When the command is executed from the selected PG, a new operation group is added to the program.


When the command is executed from an operation group in the OLP Program dashboard, the PG has to be selected afterward. A new operation group is then added to the program.


When the command is executed from an operation in the OLP Program dashboard, the PG has to be selected afterward. The new operation is then added to the operation group.
