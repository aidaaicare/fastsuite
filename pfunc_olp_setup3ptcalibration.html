3 Point Calibration

Setup workpiece

 


With the 3Point calibration command, the workpiece setup can be calibrated to its real position in the workcell.

After the command has been started, a file browser appears. A file with the calibration coordinates can be searched for and selected. This file is either of the type .pts or .txt. The content of the file will be loaded.

Then the three corresponding points (on the workpiece) in the 3D View need to be selected subsequently. The confirmation symbol appears when succeeded.
