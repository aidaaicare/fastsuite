The FASTSUITE Edition 2 Console is used to inform the user about any situation that either informs or warns about the actual status and results of his interactions. The button opens the console log window.



There are too many possible messages that can appear in the console. It is therefore impossible to describe them all in this documentation. But basically these messages are divided in three categories:

INFO

Information only. These messages do not report any problem or other issue.

WARNING

The message is a warning only, informing about situations that might lead to unexpected results or behaviors. A warning situation does not stop or block the intended activity.

ERROR

The message reports about a problem of such matter that it cannot execute or terminates the intended activity. The user needs to solve the problem to be able to continue this activity.

To inform the user that there are new messages in the console, the console button at the bottom will be flashing the corresponding warning or error symbol, like in the example below.

The window is of the style dashboard and can be dragged to any desired location on screen and can be re-sized to any preferred size.
