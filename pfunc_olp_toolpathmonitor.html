Toolpath monitor

OLP


The Toolpath monitor shows the active program in its sequence of running. For each operation (group) all the steps, i.e. all teachable positions, are displayed with their properties, as shown in the example below.


The monitor is a table with several columns.

Name

The name of the program, program call, operation group, operation or toolpath element (tag name).

Each program, operation group or operation can be expanded or collapsed by clicking on the triangle marker in front of the name.

Issues

Informs for potential problems in the program. In case of an issue, for example a collision, reachability or singularity problem, a red issue symbol is shown.

 

Hovering the mouse over the issue symbol will display its content in the 3D View.

Configuration

The configuration of the resource at that tag position.

Target type

The target type for the tag position. This can be a Joint or Cartesian type.

Motion type

The motion of the resource for the tag position. The system supports three different motion types.

Linear

PTP

Circular

 

Pressing the motion icon opens the selector and enables to change it here.

 

Process type

The process specific information of the tag position.

Feed rate

The feed rate with which the tool frame moves along the tool path.

Before and After events

Both columns show the applied events for the tag position.

The dashboard supports cross highlight between the items in the dashboard and the toolpath elements in the 3D View.


A Pie menu can be opened in the dashboard at each tag position. It includes some tag edit commands.

Activate Manipulator

Cut selection

Paste tag

Remove tag

Event

Suppress

Copy tag

While simulating a program, the actual position during the simulation is highlighted in this dashboard as well.
