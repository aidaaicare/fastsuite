Lock layout

Build layout


The Lock layout command freezes all the existing adapter connections of the layout against some (accidental) manipulation, like dragging the (child) component away.

From that status onwards, new adapter connections still can be created. And because they are not affected by the lock, these connections can be manipulated freely. To include them in the lock state, the command has to be applied again.
