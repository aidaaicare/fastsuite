Save as

File Manager


To save the component document. It opens a standard file browser to set the path where to store the document and under what name. The type of document is predefined already.


When the command is executed on a parent component a message appears when a child component is unsaved too. It has to be decided if the unsaved children components have to be stored simultaneously. This can be confirmed or rejected. When confirmed all children will be stored in the same folder on disk as the parent component.
