Fixture property defaults

Fixture Builder


The Fixture default parameter command opens the dashboard with the default design dimensions and other properties of the plates. The values that are set in this dashboard will be used to initially generate the fixture.


General parameters

This container opens the general properties of the fixture. The editor shows some base dimensions that apply to all plates or the global design of the fixture.


Attribute

 

Description

Remark

Fixture type

 

The base design of the fixture. Select from.


Base plate thickness

The thickness of the base plate.

Value ≥ 1,0 mm.

Support plate thickness

The thickness of all support plates.

Value ≥ 1,0 mm.

Offset to workpiece

The offset (distance) of the support plate contour from the workpiece.

Value ≥ 0 mm.

Corner radius

The corner radius at the support plate between the top contour and vertical sides.

Value ≥ 0 mm.

Plate design

This container includes all the dimensions of the support plate.


Attribute

 

Description

Remark



Inner corner radius

The inner radius of the support plate stanchion.

Value ≥ 0 mm.

Outer corner radius

The outer radius of the support plate stanchion.

Plate height ≥ value ≥ 0 mm.

Minimum support plate height

The (minimum) height of the support plate stanchion.

Value ≥ 1,5 * height of engraving text.

Min. value = 15 mm.

Extend stanchions

 

Activation to extrapolate the workpiece support shape of the plate.

Out rim angle

The maximum local angle at the workpiece outline to allow the contour extrapolation.

The angle is measured between the base plate and the workpiece outline. If the measured angle is smaller than the out rim angle, only then the extrapolation is applied.

30 deg ≤ value ≤ 100 deg.

Shortening length

Shortens the support shape of the plate from the outline inwards, but only if the outline angle is larger than the out rim angle.

Value ≥ 0 mm.

Extension length

The length of extrapolation, but only if the outline angle is smaller than the out rim angle.

Value ≥ 2 * size of relief.

Total extension length

The length of the plate stanchion.

Value ≥ 0 mm.

Is measured from the outline position by default. When an outline extrapolation has been applied, the length is measured from the extrapolated outline.

Maximum height of extension

The maximum height of the outline extrapolation.

Value ≥ 0 mm.

This dimension is prior to the extension length, i.e. if the extension length exceeds the maximum height, it will be limited to this height.

Extend support plate to base plate boundary

The support plate stanchion is extended up to the edges of the base plate.

Only applies when the extension length of the stanchion is set > 0 mm.

Slot / interlocking

This container includes all the dimensions and settings of interlocking the support plates and fitting them on the base plate.

Support plate interlocking

Support plate to base plate


Attribute

 

Description

Remark

Slot corner type

 

The corner shape of the interlocking slot. Four predefined shapes can be selected.

See below the explaining pictures of the different shapes.

Slot percentage

The percentage of height of the support plate to cut out the slot shape.

10 deg ≤ value ≤ 90 deg.

For the X support plate the slot is cut from the bottom side (i.e. base plate contact side) upwards.

Slot corner radius

The radius of the slot corner.

Value ≥ 0 mm.

Slot offset

The contact offset between two plates.

Value ≥ 0 mm.

Slot tolerance

The gap between two plates at the sides of the interlocking.

Value ≥ 0 mm.

Reverse slot direction

Reverses the cut out direction of the slots of the interlocking support plates.

Tab length

The length of the support plate tabs to fit in the base plate.

Value ≥ 2 * support plate thickness.

The default height of the tab is equal to the base plate thickness.

Tab height cutback

To reduce the tab height.

0 ≤ value ≤ 0,8 * base plate thickness.

Tab distance from edge

The distance of the first tab to the edge of the support plate.

Value ≥ 0 mm.

Maximum number of tabs

 

The maximum amount of tabs that will be created on each support plate.

Tabs are equally distributed.

Minimum distance between tabs

 

The minimum allowed distance between the tabs.

This parameter overrules the max. number of tabs parameter.

No cross interlockings

 

Option to prevent that tabs of interlocked support plates intersect each other.

Engraving

Weight reduction

This container includes all the dimensions and settings for the weight reduction of the support and base plates.


Relief

This container includes all the dimensions and settings to create workpiece relief areas for all the support plates.
