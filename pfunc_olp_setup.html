Setup

OLP


With the Setup command the minimum, and mandatory, configuration is set to be able to start any offline programming. It consists of setting the two major coordinates systems: the tool frame and base frame. And furthermore to assign the manufacturing technology, if that has not been done already at resource level.

When the command has been started, the already defined base and tool frames are highlighted.


From here the configuration and other related can be defined.


Base frame and tool frame

At both the base and tool frame types a Pie menu can be called to edit the frame or frame setup.

The tool frame Pie menu is identical to the base frame Pie menu that is shown above.


Manufacturing technology

For the controller and its resources the manufacturing technology and its programs can be managed here. On the controller or resource component a Pie menu opens.


It is recommended to set the technology at the tool resource that is going to execute the manufacturing process.
