Display adapters

Display filter


Adapters are intelligent blocks of information indicating the contact position at a resource, where other resources easily can attach to, to build a complete manufacturing equipment device. This adapter has a graphical representation. For convenience the adapter visualization can be switched off with the Show all adapters button. In contradiction to the Swap Show/Hide operation, this command does not switch between workspaces, nor does it move the adapter between workspaces. It only displays or hides all the present adapters in the current workspace, irrelevant if other objects, such as the attached geometry, are shown or hidden.

In the Adapter class selection options the selection for displaying or hiding the adapters is separated between electrical and mechanical adapters. Thus, they can be visualized independently from each other.
