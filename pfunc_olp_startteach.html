Start teach

Program toolpath


With the Start teach command a toolpath can be generated through single manual steps, where the resource is teached to its wanted position(s). Process geometry is not required to do this.

When the command is executed, the active tool frame is highlighted and the interactive manipulator is placed on top of this tool frame. Like in the example below.


With the interactive manipulator the tool frame, and the resources to which it is defined, can be positioned at the desired teach position. After that the Pie menu has to be called at the Manipulator's center to insert that location as a new toolpath position.
