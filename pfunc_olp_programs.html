Program manager

OLP


For the active manufacturing setup (resources and controller) multiple programs can be created for which tool paths can be generated. With the Program manager dashboard, these programs are managed.


Active controller

In this container the current active controller is displayed for which the programs are generated. If the layout has multiple manufacturing resources with the individual controllers, like in a workcell, the active controller can be set here by selecting the wanted one from the drop down list.


Programs

In this container the programs (not their content) are managed.


A list of available (created and / or loaded) programs is shown, where the green marked program is the currently active one. Simply selecting another program makes that one active for generating tool paths or other viewing or editing activities.

A simple left button mouse click on the active program opens the name editor to change the name if wanted.

Programs are of the type Main or Sub. Sub programs can represent a program sequence that will be part of a main program. In the Active program dashboard they can be loaded into an active main program. Once a sub program has been included into another program it cannot be set to the type Main. Switching the program type is done with the state control .

The Program properties container shows some information about the active program. The Type attribute tells about the program status.
