Contour based process geometry

Process geometry


With the Create contour process geometry command, this type of process geometry can be created and edited.

Create

Creating the process geometry from contour elements automatically starts the Contour search functionality of FASTSUITE Edition 2. It supports in picking those curve elements that are going to build the required contour for the manufacturing operation.

The contour search supports three different modes to come to a result. The Pie menu has to be called in the 3D View to set the active search mode.

After confirmation the process geometry is displayed and can be recognized by several colored symbols.

Process geometry symbols

The selected contour elements for the process geometry are colored in light blue.

A white ball with red arrow, indicating the start position of the contour and its direction (for search).

Purple arrows propagated over the contour elements, indicating the normal vector on the contour.

When a normal vector is picked a white 3D arrow appears to indicate this normal vector will be edited. The white ball is also placed on this normal vector.

From here the process geometry creation can be completed. On the white normal vector symbol a Pie menu can be opened.

Then the contour based process geometry has been created.


When the contour can be interpreted as some sort of primitive, regular shape, like a rectangle, circle, etc., the process geometry creation will recognize that. At computation the process geometry will be included with a frame in the center of the shape. A text will be displayed what shape has been detected. The regular shape recognition will only be applied when the option has been activated in the settings of the contour search.

 


Edit

An existing contour based process geometry object always can be modified, but only if the corresponding process geometry command has been started. Then the Edit Pie menu can be opened on a process geometry object when hoovering over that object.
