Setup technology

Setup manufacturing


The Technology command opens a small window with all technology packages available in the running session of FASTSUITE Edition 2. The currently assigned technology of the resource or controller is highlighted.


A different technology can be selected here, after which the windows automatically closes. In the background all package related information is loaded and will appear in the divers commands and dashboards. The technology has been set for all the components (resources and controller) of the manufacturing chain.
