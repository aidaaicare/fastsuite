Save layout

Layout Builder


With this Save layout command from the function toolbar the layout will be stored on disk as a layout type .cenlay file. A file browser will be opened to define the location where to store the file. A name has to be given to the layout.


Furthermore the command supports to possibility to save the document in another (CAD typical) format. The other format can be selected from the drop down list in the option Save as type. When applying one of the other CAD formats, the save operation will store all the resources and controllers from the layout.
