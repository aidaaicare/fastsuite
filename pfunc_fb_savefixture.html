Save fixture

Fixture Builder

 

With this Save fixture command from the function toolbar the fixture will be stored on disk as a resource type .cenres file, ready for re-use in a layout or as a template for other workpieces. A file browser will be opened to define the location where to store the file and under what name. At creation a system default name is given, but it can be modified before saving the document.

Furthermore the command supports to possibility to save the fixture in another CAD typical format. The format can be selected from the drop down list in the option Save as type.
