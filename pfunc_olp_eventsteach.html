Events and teach

OLP


To manipulate the motion and behavior of the resource along an initially computed toolpath the Events and teach command is used.

When the command is started, the interactive teach Manipulator is placed at the active tool frame.



With or without the use of this Manipulator now the toolpath elements, their position and behavior can be modified. Several functions will become available.


Hovering over a failure toolpath element will open a message window with information about the failure, like in the example below.



When hovering over a toolpath element, the Pie menu can be opened.
