Move the controller

Edit controller


With the Move command the position of the object (adapter, frame, etc.) can be modified.


When the command is started, the Move Manipulator is placed at the object, like in the example below. The object will now move along with the Manipulator. The move operation remains active until the Manipulator is closed by calling the Pie menu at the Manipulator's center sphere and picking the Pie center, by picking in the empty space of the 3D View or by pressing the key.


Making use of the Manipulator's interactive functionality or coordinates window enables to exactly place the object at the required location and orientation.

Moving the controller with this command should not be confused with moving a controller in a layout, where the whole controller (document) is repositioned in the context of the global space of that layout.
