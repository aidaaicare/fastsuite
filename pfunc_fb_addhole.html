Add fixing hole

Edit fixture

 

With the Add fixing hole command new holes can be created in the base plate or support plate. When the command is executed, a small window appears to enter the hole's diameter.

 

The new hole is created at the Manipulator's center. The hole's reference circle is created on the Manipulator's UV-plane.

 

Position

Added hole

 

Based upon the principle that the mounting hole usually has to remain fixed, regardless the size of the plate, its initial creation uses the Manipulator's position. It has no positional link to the plate design.
