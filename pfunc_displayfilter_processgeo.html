The display of the process geometry has an extra filter option. The process geometry can be filtered for their available attributes that are listed. Simply grab an attribute from the list and drag it to the filter section of the UI. The filter is then applied automatically. Filters of multiple attributes is possible. Dragging a new attribute to an existing filter will generate an AND operation of the attributes.

The filter can be activated or de-activated by setting the global check marker.

The attributes within a filter can be applied, by setting the check marker, or inverted, by un-setting the check marker. The inverted attribute filter shows the complement of the applied result.

With the button the current filter can be terminated, after which all process geometries are shown again.

The Render process geometry on top option is a possibility to always show the process geometry elements on top of the screen in any camera view direction, regardless if they would have been covered by any other geometry in front of the camera view.
